Fling Engine is a cross platform Vulkan game engine with a focus on providing a simple to use data oriented framework.

Features

  • Cross platform (Windows and Linux)
  • Deferred renderering pipeline
  • Effecient gaemplay framewok via [Entt](https://github.com/skypjack/entt)

Get the source code on GitHub

Useage

All you need to do in order to make a game with the Fling Engine is to override the `Fling::Game` class, and specify your game when you create your engine instance. Your game's `main.cpp`: 

    #include "FlingEngine.h"

    /* Entry point for using the Fling Engine! */
    int main(int argc, char* argv[])
    {
      Fling::Engine Engine = {};
    
      try
      {
        Engine.Run();
      }
      catch (const std::exception& e)
      {
        std::cerr << e.what() << std::endl;
        return EXIT_FAILURE;
      }
    
      return EXIT_SUCCESS;
    }
